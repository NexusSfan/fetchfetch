# `fetchfetch`

Fetch information about your `*fetch` tools.

## Example

```console
$ fetchfetch
 ______________________________________
|                                      | Fastfetch: 2.33.0
|  >_                                  | Neofetch: 7.1.0
|                                      | onefetch: 2.22.0
|                                      |
|                                      |
|                                      |
|                                      |
|                                      |
|______________________________________|
```
